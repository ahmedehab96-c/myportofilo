## Build & Deploy

```bash
chmod +x build_web.sh run_web.sh
./build_web.sh
./run_web.sh   # http://localhost:8080
```

**Netlify:** push to `main` → `scripts/netlify_build.sh` builds `build/web` (~25 MB).

```bash
NETLIFY_DEPLOY=1 ./build_web.sh
```
